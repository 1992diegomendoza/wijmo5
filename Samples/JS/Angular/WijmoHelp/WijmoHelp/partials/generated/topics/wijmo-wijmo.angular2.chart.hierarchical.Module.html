Module wijmo/wijmo.angular2.chart.hierarchical

File
wijmo.angular2.js

Contains Angular 2 components for the wijmo.chart.hierarchical module.

wijmo.angular2.chart.hierarchical is an external TypeScript module that can be imported to your code using its ambient module name. For example:

import * as wjHierarchical from 'wijmo/wijmo.angular2.chart.hierarchical';
 
@Component({
    directives: [wjHierarchical.WjSunburst],
    template: `
      <wj-sunburst [itemsSource]="data" [binding]="'y'" [bindingX]="'x'">
      </wj-sunburst>`,
    selector: 'my-cmp',
})
export class MyCmp {
    data: any[];
}

WjSunburst

Angular 2 component for the Sunburst control.

Use the wj-sunburst component to add Sunburst controls to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjSunburst component is derived from the Sunburst control and inherits all its properties, events and methods.

The wj-sunburst component may contain WjFlexChartLegend and WjFlexPieDataLabel child components.
